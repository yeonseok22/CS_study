# 알고리즘 문제

## 스터디 안내 사항
- 디렉터리 이름 형식 : ```코테사이트_문제번호및제목```
  - 풀이 파일 이름 형식 : ```코테사이트_문제번호및제목.확장자```
  - 코테 사이트
    - 백준 : bj
    - 코드업 : cu
    - 프로그래머스 : pro
    - 책 : book
    - 기타 : etc
  - 문제 제목이 길다면 간단하게 작성
  - 예시 : 백준 1001번 문제 javascript로 품 => ```bj_1001.js```

- 파일에 [문제 주소], [풀이 방법], [푼 사람] 등을 주석으로 간단히 작성 부탁드립니다.
- 궁금한 사항이 있다면 연락주세요
